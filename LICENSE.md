## 🔧 Common Issues & Fixes

**❌ Rectangle isn't snapping windows**
> Make sure you've granted **Accessibility permissions** in System Settings → Privacy & Security → Accessibility. Rectangle needs this to control window positions.

**❌ Keyboard shortcuts aren't working**
> Check if another app is using the same shortcuts. You can customize all shortcuts in Rectangle's Preferences.

**❌ "Rectangle cannot be opened because it's from an unidentified developer"**
> This is a macOS Gatekeeper warning. Right-click (or Control-click) the app and select **"Open"** — then confirm you want to open it. This only happens the first time.

**❌ Windows snap to the wrong display**
> Make sure your displays are arranged correctly in System Settings → Displays → Arrangement. Rectangle respects your display layout.

**❌ The app asks for permissions every time I launch**
> This is a macOS bug. Remove Rectangle from the Accessibility list, restart the app, and re-add it when prompted.
